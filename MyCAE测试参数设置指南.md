# MyCAE 测试参数设置指南

## 1. 使用说明

本文档用于说明 MyCAE 功能测试和课堂演示时，各类参数应该如何填写。建议优先使用一个简单、稳定、容易解释的结构算例：

```text
长方体悬臂梁结构静力分析
```

该算例的好处是：

- 几何简单，老师容易理解；
- 固定端和受力端很容易通过面拾取定义；
- Gmsh 网格容易生成；
- CalculiX 静力分析流程清晰；
- 结果可以展示位移和 Von Mises 应力。

推荐单位体系：

```text
长度：mm
力：N
应力 / 弹性模量：MPa，也就是 N/mm^2
```

在该单位体系下，如果弹性模量使用 `210000 MPa`，几何尺寸使用 `mm`，载荷使用 `N`，整体是自洽的。

| 高度 Height | 40 | 梁的高度，沿 Z 方向 |

## 5. 面拾取与面组参数设置

悬臂梁分析需要至少两个面组：

```text
Fixed_End：固定端
Load_End：受力端
```

### 5.1 固定端面组

菜单路径：

```text
几何 -> 拾取面
```

操作步骤：

1. 在三维窗口中点击长方体左端面。
2. 点击：

```text
几何 -> 面组 -> 从拾取创建面组
```

推荐参数：

| 参数 | 推荐值 |
|---|---|
| 面组名称 | Fixed_End |
| 是否物理组 | 是 |
| 局部网格尺寸 | 0 或不启用 |

物理含义：

```text
Fixed_End 表示悬臂梁左端固定面。
```

预期结果：

```text
工程树中出现 Fixed_End；
属性面板显示该面组包含 1 个面；
后续可作为结构约束目标。
```

### 5.2 受力端面组

操作步骤：

1. 开启拾取面模式。
2. 点击长方体右端面。
3. 从拾取创建面组。

推荐参数：

| 参数 | 推荐值 |
|---|---|
| 面组名称 | Load_End |
| 是否物理组 | 是 |
| 局部网格尺寸 | 0 或不启用 |

物理含义：

```text
Load_End 表示悬臂梁右端加载面。
```

### 5.3 局部网格尺寸参数

如果要展示“面组局部网格控制”，可以对 `Load_End` 设置局部网格尺寸。

菜单路径：

```text
几何 -> 面组 -> 设置面组局部网格尺寸
```

推荐参数：

| 参数 | 推荐值 |
|---|---:|
| Local mesh size | 8 |

说明：

```text
设置为 8 mm 后，Load_End 附近网格会更细。
如果只是想稳定生成网格，可以先不启用局部网格。
```

## 6. 网格参数设置

菜单路径：

```text
仿真 -> 生成网格
```

### 6.1 推荐稳定网格参数

这是最推荐用于课堂演示的一组参数：

| 参数 | 推荐值 |
|---|---|
| 单元类型 | 一阶四面体 Tetra4 |
| 自动尺寸 | 关闭 |
| 最小尺寸 | 8 |
| 最大尺寸 | 20 |

说明：

```text
这个尺寸对 200 x 40 x 40 mm 的梁比较稳定；
网格数量不会太大，生成速度较快。
```

预期结果：

```text
生成 .msh 文件；
工程树中出现网格对象；
日志显示节点数和四面体数量。
```

| 最小尺寸 | 5 |
| 最大尺寸 | 12 |


## 7. 材料参数设置

菜单路径：

```text
工况 -> 结构工况 -> 创建结构材料
```

### 7.1 钢材料参数

推荐用于悬臂梁演示：

| 参数 | 推荐值 | 单位 |
|---|---:|---|
| 材料名称 | Steel | - |
| 弹性模量 Young's Modulus | 210000 | MPa |
| 泊松比 Poisson's Ratio | 0.30 | - |
| 密度 Density | 7.85e-9 | tonne/mm^3 或按界面单位填写 |

如果界面没有密度项，可以只填写：

```text
Young's Modulus = 210000
Poisson's Ratio = 0.30
```

说明：

```text
在 mm-N-MPa 单位体系中，210000 MPa 是常用钢材弹性模量。
```

### 7.2 铝合金材料参数

用于对比测试：

| 参数 | 推荐值 | 单位 |
|---|---:|---|
| 材料名称 | Aluminum |
| 弹性模量 Young's Modulus | 70000 | MPa |
| 泊松比 Poisson's Ratio | 0.33 | - |

预期现象：

```text
同样载荷下，铝合金位移通常大于钢材。
```

菜单路径：

```text
工况 -> 结构工况 -> 创建结构约束
```

### 9.1 固定约束

推荐参数：

| 参数 | 推荐值 |
|---|---|
| 名称 | Fixed_Support |
| 类型 | Displacement / Fixed |
| 目标几何 | Box_1 或 Beam_1 |

| Uz | 勾选，值 0 |


## 10. 结构载荷参数设置

菜单路径：

```text
工况 -> 结构工况 -> 创建结构载荷
```

### 10.1 端面向下载荷

推荐参数：

| 参数 | 推荐值 |
|---|---|
| 名称 | Downward_Load |
| 类型 | Force / Surface Load，按界面已有类型选择 |
| 目标边界条件或目标面组 | Load_End |
| 方向 X | 0 |
| 方向 Y | 0 |
| 方向 Z | -1000 |
| 单位 | N |
| 启用 | 是 |

如果当前界面是单值载荷字段：

| 参数 | 推荐值 |
|---|---|
| Field Name | Fz 或 load |
| Value | -1000 |

物理含义：

```text
在梁右端施加向下的 1000 N 载荷。
```

说明：

```text
如果坐标系中 Z 轴向上，则负 Z 表示向下；
如果模型方向不同，可以根据界面显示调整方向。
```

### 10.2 较小载荷测试参数

如果担心变形过大，可以使用：

| 参数 | 推荐值 |
|---|---:|
| Fz | -100 |

### 10.3 较大载荷测试参数

如果想让变形效果更明显，可以使用：

| 参数 | 推荐值 |
|---|---:|
| Fz | -5000 |

注意：

```text
载荷过大可能导致结果显示变形夸张；
```

## 12. CalculiX 求解参数设置

菜单路径：

```text
仿真 -> CalculiX
```

或菜单中显示为具体求解器名称，例如：

```text
仿真 -> CalculiX - staticStructural
```

运行前检查：

```text
1. 已创建几何
2. 已生成网格
3. 已创建结构材料
4. 已创建材料分区
5. 已创建固定约束
6. 已创建结构载荷
7. CalculiX 路径有效
```

预期生成文件：

```text
solver/calculix/某个算例目录/*.inp
solver/calculix/某个算例目录/*.dat
solver/calculix/某个算例目录/*.sta
solver/calculix/某个算例目录/*.frd
solver/calculix/某个算例目录/*.log
```

如果求解失败，优先检查：

```text
是否有网格
是否有材料
是否有材料分区
是否有固定约束
是否有载荷
面组是否设置为物理组
CalculiX 路径是否正确
```

## 13. 结果后处理参数设置

### 13.1 结果字段选择

推荐演示顺序：

```text
Displacement Magnitude
-> Ux
-> Uy
-> Uz
-> Von Mises Stress
```

说明：

```text
先看位移模量，直观展示梁的整体变形；
再看 Von Mises 应力，展示固定端附近应力集中。
```

### 13.2 变形比例设置

推荐参数：

| 场景 | 变形比例 |
|---|---:|
| 默认观察 | 1 |
| 变形不明显 | 5 |
| 演示夸张变形 | 10 |
| 查看真实未变形位置 | 0 |

建议演示：

```text
先用 1x；
如果变形不明显，再调到 5x 或 10x；
同时打开未变形轮廓。
```

### 13.4 未变形轮廓

## 17. 下午演示推荐流程

最稳流程：

```text
1. 打开样例工程 box_pressure_demo
2. 展示工程树
3. 选择几何并显示几何边线
4. 展示面组和物理组
5. 显示网格
6. 切换网格透明显示
7. 展示材料、约束、载荷
8. 选择已有结果
9. 显示位移云图
10. 切换 Von Mises 应力
11. 调整变形比例为 5
12. 显示网格边和未变形轮廓
13. 展示极值和点选查询
14. 导出截图或报告
15. 说明 CalculiX 和 OpenFOAM 插件架构
```

如果现场时间充足，再演示从零创建长方体和生成网格；如果时间有限，优先展示样例工程完整流程。
